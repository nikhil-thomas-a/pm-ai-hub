# ⚡ PM AI Hub

> Practical AI workflows for Delivery Program Managers — real prompts, live tools, and case studies.

🌐 **Live site:** `https://nikhil-thomas-a.github.io/pm-ai-hub`

---

## What Is This?

PM AI Hub is a portfolio and learning resource built by a Delivery Program Manager to help other PMs use AI effectively in their day-to-day workflows.

Most AI tools are built *for* developers or *by* developers. This is built by someone who runs standups, writes escalations at 11pm, and has sat through 47 sprint retrospectives.

---

## Three Pillars

### 🎓 Prompt Playbook
Copy-paste prompts for the three biggest PM pain points:
- **Stakeholder Reporting** — Turn messy sprint data into exec-ready updates
- **Escalation & Incidents** — Draft escalations and post-mortems under pressure
- **Sprint & Delivery Tracking** — Spot risks early, run better retros

Each prompt includes: difficulty level, time estimate, the exact prompt, pro tips, and a step-by-step guide.

### ⚙️ Workflow Lab
Live AI-powered tools PMs can use right now:
- **Slack Summarizer** — Reads a Slack channel and extracts blockers, decisions & action items

### 📖 Case Study Journal
Real delivery problems, exact prompts used, measurable outcomes.

## Tech Stack

- React 18 + Vite
- Claude API (Anthropic) for AI features
- Slack MCP for live Slack integration
- GitHub Actions for CI/CD
- GitHub Pages for hosting

---

## Run Locally

```bash
git clone https://github.com/nikhil-thomas-a/pm-ai-hub.git
cd pm-ai-hub
npm install
npm run dev
```

Open `http://localhost:5173`

## Deploy

Every push to `main` auto-deploys via GitHub Actions. No manual steps needed.
